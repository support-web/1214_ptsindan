# 投資診断サイト生成プロンプトテンプレート

## 基本プロンプト（コピペ用）

以下のプロンプトをコピーして、【】内を埋めてAIに渡してください。

---

```
添付の「投資診断サイト構築仕様書」に基づいて、投資診断サイトを作成してください。

## ターゲット
【ここにターゲットを詳しく記入】

## 要件
- デザインテーマ：【テーマIDまたは「ターゲットに最適なものを選択」】
- 診断タイプの命名：【動物型/職業型/キャラクター型/色・宝石型/食べ物型/属性型 から選択、または「ターゲットに最適なものを提案」】
- 質問数：【5〜10問の範囲で指定】
- 特に重視するポイント：【例：SNSシェアしやすさ、信頼感、親しみやすさ など】

## 出力形式
- React (JSX) 単一ファイル
- Tailwind CSS使用
- 完全に動作するコード
```

---

## ターゲット記入例

### 例1：若い女性向け
```
20代後半の女性。投資は全くの未経験で、「投資＝難しい・怖い」というイメージを持っている。
推し活（アイドル、アニメ）が趣味で、グッズやライブにお金を使いがち。
将来への漠然とした不安はあるが、具体的な行動には移せていない。
SNSはInstagramとXをよく使う。かわいいものが好き。
```

### 例2：若手ビジネスマン向け
```
25〜30歳の男性会社員。年収400〜500万円程度。
NISAという言葉は知っているが、まだ始めていない。
「そろそろ投資を始めないと」という焦りはある。
効率的に情報を得たい。長い説明より、要点をサクッと知りたいタイプ。
ゲームやガジェットが好き。
```

### 例3：子育て世代の主婦向け
```
30代後半の主婦（パート勤務）。子供の教育費や老後資金が心配。
夫の収入だけでは不安で、自分でも資産形成を考え始めた。
投資は「損をしそうで怖い」というイメージ。
家計管理はしっかりしている。堅実派。
LINEをよく使う。派手なものより落ち着いたデザインが好み。
```

### 例4：シニア層向け
```
55〜65歳の男女。退職金の運用や年金だけでは不安を感じている。
投資経験は株を少しやったことがある程度。
「今さら始めても遅いのでは」という気持ちもある。
信頼できる情報を求めている。派手なデザインより落ち着いた雰囲気が好み。
文字は大きめで読みやすいほうがいい。
```

### 例5：Z世代・大学生向け
```
18〜22歳の大学生。バイト代の一部を投資に回してみたいと思っている。
TikTokやYouTubeで投資系の動画を見たことがある。
少額から始められることに興味あり。
ゲーム感覚で楽しめるものが好き。
「映える」「エモい」ものに惹かれる。
```

## カスタマイズオプション

### デザインテーマ指定

```
デザインテーマ：cute
```
↓ 以下から選択
- `cute` - キュート・ポップ（10-20代女性向け）
- `minimal` - ミニマル・クリーン（20-30代男女向け）
- `premium` - プレミアム・ラグジュアリー（高収入層向け）
- `friendly` - フレンドリー・カジュアル（全般向け）
- `professional` - プロフェッショナル（ビジネス層向け）
- `playful` - ゲーミフィケーション（若年層・ゲーム好き向け）
- `natural` - ナチュラル・オーガニック（30-40代女性向け）
- `retro` - レトロ・ノスタルジック（30-40代向け）

### 診断タイプ命名指定

```
診断タイプの命名：動物型（かわいい系の動物で）
```
↓ 以下から選択・アレンジ
- 動物型：ライオン、カメ、フクロウ、リス など
- 職業型：トレーダー、銀行員、研究者、起業家 など
- キャラクター型：勇者、魔法使い、賢者、商人 など
- 色・宝石型：ゴールド、シルバー、エメラルド、ルビー など
- 食べ物型：ステーキ派、煮込み派、サラダ派 など
- 属性型：火属性、水属性、風属性、地属性 など

### 重視ポイント指定

```
特に重視するポイント：SNSでシェアしたくなるデザイン、結果画面の華やかさ
```
↓ 例
- SNSシェアしやすさ
- 信頼感・安心感
- 親しみやすさ
- ゲーム感覚の楽しさ
- 高級感・特別感
- わかりやすさ・シンプルさ
- データの説得力

## 完全版プロンプト例

```
添付の「投資診断サイト構築仕様書」に基づいて、投資診断サイトを作成してください。

## ターゲット
20代後半の女性。投資は全くの未経験で、「投資＝難しい・怖い」というイメージを持っている。
推し活（アイドル、アニメ）が趣味で、グッズやライブにお金を使いがち。
将来への漠然とした不安はあるが、具体的な行動には移せていない。
SNSはInstagramとXをよく使う。かわいいものが好き。

## 要件
- デザインテーマ：cute
- 診断タイプの命名：動物型（かわいい系の動物：うさぎ、ねこ、ペンギン、ハムスターなど）
- 質問数：7問
- 特に重視するポイント：SNSでシェアしたくなる可愛いデザイン、ネガティブな結果を出さない、投資への心理的ハードルを下げる

## 追加要件
- 質問文は「〜かな？」「〜だよね？」などカジュアルな口調で
- 結果画面には「推し活資金を増やすコツ」セクションを追加
- パステルカラーを基調に
- 絵文字を効果的に使用

## 出力形式
- React (JSX) 単一ファイル
- Tailwind CSS使用
- 完全に動作するコード
```

## トラブルシューティング

### 生成されたコードが動かない場合
```
生成されたコードにエラーがあります。以下を修正してください：
- [エラー内容を貼り付け]
```

### デザインを調整したい場合
```
以下の点を修正してください：
- 背景色をもう少し明るく
- ボタンをもう少し大きく
- フォントサイズを全体的に大きく
```

### 質問内容を変更したい場合
```
質問3を以下の内容に変更してください：
「〇〇〇〇？」
選択肢：
A. 〇〇〇
B. 〇〇〇
C. 〇〇〇
D. 〇〇〇
```
